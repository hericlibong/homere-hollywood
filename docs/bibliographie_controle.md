# Bibliographie de contrôle

Cette bibliographie rassemble les textes et ressources utilisés ou à utiliser pour vérifier le corpus *Troy*. Elle n'est pas une bibliographie académique complète : c'est une liste de contrôle pour l'édition du dataset et la future web app.

## Homère

- Homère, *Iliade*.
  - Usage : Achille, Hector, Patrocle, Priam, Briséis, grands chants de référence.
  - Contrôle : PDF local ; Perseus ; Scaife Viewer.

- Homère, *Odyssée*.
  - Usage : Ménélas et Hélène après la guerre, cheval de Troie raconté, mémoire des morts.
  - Passages utiles : Chant IV, Chant VIII, Chant XXIV.
  - Contrôle : Perseus.

## Cycle Troyen

- *Chants Cypriens*.
  - Usage : Pâris à Sparte, départ d'Hélène, causes de la guerre, rassemblement grec.
  - Statut : poème perdu, connu par résumé et fragments.

- *Éthiopide*.
  - Usage : mort d'Achille par Pâris et Apollon, avant la chute de Troie.
  - Statut : poème perdu, connu par résumé et fragments.

- *Petite Iliade*.
  - Usage : armes d'Achille, Ajax, préparation du cheval.
  - Statut : poème perdu, connu par résumé et fragments.

- *Sac d'Ilion*.
  - Usage : cheval de Troie, prise de la ville, Priam, Cassandre, Andromaque, Énée.
  - Statut : poème perdu, connu par résumé et fragments.

- *Retours*.
  - Usage : destins des chefs grecs après Troie, notamment Agamemnon et Ménélas.
  - Statut : poème perdu, connu par résumé et fragments.

Ressources de contrôle :

- Theoi Classical Texts Library, *Epic Cycle Fragments* : https://www.theoi.com/Text/EpicCycle.html
- Center for Hellenic Studies, *Epic Cycle* : https://chs.harvard.edu/primary-source/epic-cycle-sb/

## Tradition gréco-romaine conservée

- Virgile, *Énéide*, livre II.
  - Usage : cheval, Sinon, Laocoon, Priam, incendie de Troie, fuite d'Énée.
  - Contrôle : PDF local ; Perseus ; Theoi.

- Eschyle, *Agamemnon*.
  - Usage : retour et mort d'Agamemnon à Mycènes.
  - Contrôle : Theoi ; Perseus.

- Sophocle, *Ajax*.
  - Usage : folie et suicide d'Ajax après le jugement des armes.
  - Contrôle : Theoi ; Perseus.

- Euripide, *Troyennes*.
  - Usage : captives troyennes, Hécube, Cassandre, Andromaque, Astyanax.
  - Contrôle : PDF local ; Theoi ; Perseus.

- Euripide, *Hécube*.
  - Usage : Hécube, Polyxène, Polydore, deuil troyen.
  - Contrôle : Theoi ; Perseus.

- Euripide, *Andromaque*.
  - Usage : destin d'Andromaque après Troie.
  - Contrôle : Theoi ; Perseus.

- Ovide, *Héroïdes*.
  - Usage : voix poétiques d'Hélène, Pâris, Oenone, Briséis selon les besoins.
  - Contrôle : à rattacher si ces passages deviennent centraux.

- Ovide, *Métamorphoses*.
  - Usage : variantes poétiques de mythes troyens.
  - Contrôle : à rattacher si ces passages deviennent centraux.

- Stace, *Achilléide*.
  - Usage : traditions tardives autour d'Achille.
  - Contrôle : Wikisource ; Theoi pour certains extraits.

## Mythographes et sources auxiliaires

- Pseudo-Apollodore, *Bibliothèque*.
  - Usage : synthèse de traditions, variantes et généalogies.
  - Niveau : contrôle tardif, non source première.

- Hygin, *Fables*.
  - Usage : variantes tardives et listes mythographiques.
  - Niveau : contrôle tardif, à manier prudemment.

- Strabon, *Géographie*.
  - Usage : lieux, fleuves, ethnonymes, noms propres.
  - Niveau : auxiliaire.

- Pausanias, *Description de la Grèce*.
  - Usage : traditions locales, lieux et cultes si explicitement nécessaires.
  - Niveau : auxiliaire.

## Règle d'usage

Pour la web app et les textes éditoriaux :

- citer Homère, Virgile et les tragiques quand le passage est conservé ;
- dire `d'après le résumé de Proclus` pour le Cycle Troyen quand le poème est perdu ;
- éviter de transformer un détail rituel ou culturel en source principale ;
- utiliser Apollodore, Hygin, Strabon et Pausanias comme contrôles secondaires, pas comme socle narratif principal.
