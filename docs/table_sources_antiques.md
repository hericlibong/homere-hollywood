## Legende des niveaux

- A : source litteraire conservee, prioritaire pour verifier une affirmation.
- B : source perdue ou fragmentaire, utilisable avec mention explicite du statut.
- C : source mythographique tardive, utile pour variantes et syntheses.
- D : source auxiliaire pour lieux, noms propres ou traditions locales.
