## Le film de Nolan : le hook parfait

*The Odyssey* de Christopher Nolan sort le **17 juillet 2026** — dans 3 mois. Matt Damon joue Ulysse, Anne Hathaway joue Pénélope, Zendaya joue Athéna, Charlize Theron joue Calypso. Et déjà, les costumes ont déclenché des discussions sur les inexactitudes historiques avant même la sortie. Ton projet arrive exactement au bon moment.

## Évaluation de l'idée

**Ce qui est solide :**
- L'angle est intelligent. Pas "est-ce fidèle ?" (critique facile) mais "qu'est-ce qui vient d'où ?" — Iliade, Odyssée, cycle troyen, invention pure. C'est plus riche.
- Le corpus est cohérent : *Troy* (Petersen, 2004), *Ulysse* (1954, Kirk Douglas), *Troy: Fall of a City* (série BBC/Netflix), *The Odyssey* (Nolan, 2026). Deux époques, deux supports, deux épopées.
- Les textes sont dans le domaine public. L'IA peut travailler librement dessus.
- L'actualité te donne une fenêtre éditoriale claire.

**Ce qui peut te tuer :**
- Le dataset, pas la viz. Scène par scène sur *Troy* seul, c'est facilement 80-100 entrées à catégoriser. Il faut une méthodologie stricte avant de coder quoi que ce soit.
- Le scope peut exploser. Cadrer tôt : par film ou par thème ? Je recommande **par film** comme unité de base, avec une vue transversale en bonus.

## Propositions de visualisations

J'en propose **4**, hiérarchisées. Tu n'as pas besoin des 4 — même 2 bien faites suffisent.

**1. Le bandeau scène par scène** *(viz principale — proche de IIB mais différente)*
Une frise horizontale représentant le film, découpée en segments. Chaque segment = une scène ou séquence. Code couleur par source :
- 🟠 Dans l'Iliade/Odyssée (texte direct)
- 🟡 Dans le cycle troyen (Chants cypriens, Éthiopide, etc.)
- 🔵 Tradition antique mais hors Homère (Virgile, Ovide, tragiques)
- ⚫ Invention du film

Survol = titre de la scène + note + référence textuelle si applicable. C'est le cœur du projet, immédiatement lisible.

**2. La carte du cycle troyen**
Un diagramme radial ou linéaire montrant les 8 poèmes du Cycle (Chants cypriens → Iliade → Éthiopide → Petite Iliade → Sac d'Ilion → Retours → Odyssée → Télégonie). Chaque film est positionné dessus pour montrer **quelle portion du cycle il couvre**. Ça contextualise d'un coup ce que Hollywood retient — et ce qu'il ignore.

**3. Les fiches personnages**
Pour 6-8 personnages clés (Achille, Hector, Ulysse, Hélène, Pâris, Patrocle, Briséis…) : une carte "Dans Homère / Dans le film". Destin, rôle, différences majeures. Format simple, swipeable sur mobile. Ce n'est pas une viz de données mais ça ancre le storytelling.

**4. Le comparateur de films** *(vue bonus)*
Un tableau ou radar comparant les 4 œuvres sur quelques axes : % scènes à source homérique, traitement des dieux (présents / absents / réinterprétés), liberté narrative, ton (épique / psychologique / spectaculaire). Utile pour mettre Nolan en perspective par rapport à Petersen et Kirk Douglas.

## Identité visuelle

Une direction qui colle au sujet sans être un musée :

**Palette** : fond sombre (quasi noir), terracotta/ocre pour le "Homère pur", or pour les éléments du cycle, ardoise bleue pour les ajouts, gris pour les inventions. Sobre, lisible, évocateur sans être kitsch.

**Typographie** : une serif élégante pour les titres (penser *Cormorant Garamond* ou *Playfair Display*), monospace ou sans pour les données.

**Motif discret** : une frise grecque (méandre) en séparateur, ou une silhouette d'amphore en arrière-plan très atténué. Pas plus.

**Ton général** : un data essay, pas un musée virtuel. Texte court, viz au centre, navigation linéaire par film ou par thème.
